Automatically denies comment access to known spammers in the NerdTools Bad Bots database.

== Description ==

Powered by NerdTools.co.uk, this plugin will automatically and anonymously check the authors IP address against the NerdTools Bad Bots Intrusion & Spam Detection database.

If a match is detected no commenting will be allowed and a custom error message will be displayed instead, if no match is detected commenting will be allowed as normal.

More information can be found at http://www.nerdtools.co.uk/badbots/

== Installation ==

1. Upload `nerdtools_badbot_spam_defender.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Head to 'Settings' then 'NT BB Spam Defender' and check the box to enable the plugin and add a custom error message if required and hit 'Save Changes'
4. That's it you will now be protected from future known spammers

== Frequently asked questions ==
= How do I report spam authors to the database? =

We have a seperate plugin to do this called NerdTools Bad Bots Spam Reporter which will automatically report the author of any comment classed as spam.
